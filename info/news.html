Announcements

Mac FireFox/Camino Users: I've been informed that axis labels are not always showing in Mozilla-based browsers on Macs. If you're having this problem, use the "Force image-based graphs" option when logging in

All FireFox Users: If you are having problems with graph axis labels printing really large, you can log in with the "Force image-based graphs" option to bypass this FireFox bug. Update: This problem has been resolved in FireFox 2.0.
